# glTF（ジーエルティーエフ）

glTFは、3Dモデルを表示するために作られた、現在もっとも主流な**ファイル形式**（データの保存形式）の1つです。よく **「3D界のJPEG」** と例えられます 🖼️。画像におけるJPEGのように、**ファイルサイズが小さく、Webブラウザやアプリ上ですぐに読み込んで表示できる** ように設計されているのが特徴です。

アップロードされたガイドにも `GLTFLoader` （glTFローダー）を使ってVRMを読み込むコードが出てきますが、これはVRMがglTFの仕組みを利用しているためです。

ここから、どの方向で理解を深めたいですか？

1. **VRMとの深い関係**
あなたが使おうとしている「VRM」と「glTF」は、実は切っても切れない関係にあります。その「正体」を知りたいですか？
2. **なぜ「標準」になったのか**
昔からある `.obj` や `.fbx` といった他の形式と比べて、なぜ glTF がこれほどWebで普及したのか、そのメリットを知りたいですか？
3. **ファイルの中身**
少し技術的になりますが、実際にファイルの中にどんなデータ（形、色、動きなど）が入っているのかを知りたいですか？
